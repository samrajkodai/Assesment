1.Speech Detection Model and convert it into the English language

Steps for performing Speech to Text

we need to install that two libraries

  • pip install SpeechRecognition
  • pip install PyAudio
  1. import the speech recognition library
  2. initialize the recognizer,we using a google speech recognizer.
  3. we can use a audio file or microphone to give the input.for a audio file it supports wav,AIFF, AIFF-C,FLAC.

example code using audio file:

  #import library
  import speech_recognition as sr

  # Initialize recognizer class 
  r = sr.Recognizer()

  # Reading Audio file as source
  # listening the audio file and store in audio_text variable

  with sr.AudioFile('sam.wav') as source:

      audio_text = r.listen(source)

      try:

          # using google speech recognition
          text = r.recognize_google(audio_text)
          print('Converting audio transcripts into text ...')
          print(text)

      except:
          print('Sorry.. run again...')

  #import library

Q2.Sentiment Analysis from the text

For a Sentimental Analysis usecase we can use a NLP concept.

Steps:

  • import the data and libraries
  • perform EDA

In NLP we first perform text preprocessing those steps are,

  • Stopwords

    It is used to remove the meaningless words in a text.

  • Lemmatize or stemmer

    1.Lemmatize used for find the meaningful words in a sentence.

2.Stemming algorithms work by cutting off the end or the beginning of the word. 
 For example: words such as Likes, liked, likely and liking will be reduced to like after stemming.
  • Then we convert the sentences into vectors we can use any of the technique.
          1.word2vec   
          2.WordEmbedding
          3.tf-idf
          4.countvectorizer

  • after we convert the sentences into vectors we split the data into train and test

  • then we can use Machine Learning or Deep Learning to built the model.

  • for a Machine Learning the Naive Bayes Algorithm gives the good accuracy

5.Text Classification Using Deep Learning

nlp with deep learning

  • import the dataset and libraries
  • Perform EDA

In NLP we first perform text preprocessing those steps are,

  • Stopwords

    It is used to remove the meaningless words in a text.

  • Lemmatize or PorterStemmer

    1.Lemmatize used for find the meaningful words in a sentence.

2.Stemming algorithms work by cutting off the end or the beginning of the word. 
 For example: words such as Likes, liked, likely and liking will be reduced to like after stemming.

    but if we want accurate meaning of the word we need to use lemmatizer.

  • Then we convert the sentences into vectors we can use any of the technique.

            1.word2vec   
        2.WordEmbedding
        3.tf-idf
        4.countvectorizer

  • after we convert the sentences into vectors we split the data into train and test

  • after splitting The model will be build using deep learning

  • In deep learning we add a Embedding layer as a first layer

  • and we add a LSTM Layer,the LSTM Layer have a some important cells

          1.Memory Cell
          2.Forget Gate
          3.Input Gate
          4.Output Gate

  • Next we add a dense layer with sigmoid activation fuction function and adam optimizer

    1.the sigmoid activation function used to convert the values between 0  to 1
so for the classification usecases it works well.
but for derivation of sigmoid the ranges between 0 to 0.25 only

2.In this use case we use a Adam optimizer 

adam optimizer combines the Momentum and RMSProp

  • after the model fit we predict the result for test data

  • with the help performance metrics we find the confusion matrix(true positive,true negative,false positive,flase negative)

  • Next,we find the accuracy score for the model.

  • if we are facing any overfitting we can add Dropout layer to prevent the overfitting
